# Pokédex
## Description
A Pokédex is a device which holds information about pokemons.

## Features

- Pokemon name search;

## Prerequisites
- Python 3.10.12;
- tkinter library;
- PIL library;
- requests library;
- threading library;
- urllib.request library;

## Usage

`$ python3 main.py`
